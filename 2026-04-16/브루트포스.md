# 브루트 포스란?
가능한 모든 경우의 수를 무식하게 일일이 탐색하여 100% 정답을 찾아내는 완전 탐색 알고리즘이다

반복문과 재귀 BFS, DFS등을 사용하여 구현하며 데이터가 작을때는 확실한 답을 찾아내지만 데이터가 클 때는 시간복잡도가 급격하게 커지는 단점이 있습니다

## 원리
 문제를 해결할 수 있는 모든 영역을 탐색하여, 조건에 부합하는 결과만 가져옵니다
 > ### 4자리 비밀번호 자물쇠
 > 0000부터 9999까지 하나하나 돌려가면서 자물쇠의 비밀번호를 찾는 방법
 > ### 소수 찾기
 > 1부터 N까지의 모든 수에 대해 1과 자기 자신 외에 나누어떨어지는 수가 있는지 하나씩 확인
 > ### 최대/최소 찾기
 > 배열의 모든 요소를 1대1로 비교
 
 ## 장점
 예외 없이 정확한 답을 보장하며, 구현이 직관적이고 매우 간단합니다

 ## 단점
 경우의 수가 많을 경우 속도가 느려 시간 초과가 발생할 수 있습니다

 ## 활용
 데이터의 크기가 작은 경우, 암호 해독(무차별 대입), 조합/순열 생성, 코딩 테스트 등

 ## 구현 예시
 ```
 arr = ['A', 'B', 'C']
N = len(arr)

my_comb = []
for i in range(N - 1):
    for j in range(i + 1, N):
        my_comb.append((arr[i], arr[j]))

print(my_comb)

출력:[('A', 'B'), ('A', 'C'), ('B', 'C')]
```
다중 반복문을 활용하여 조합을 만들었습니다 3개의 조합을 구하기 위해서는 반복문을 3개 사용하고 4개의 조합을 구하기 위해서는 반복문을 4개를 사용하면 됩니다 더욱 많은 조합을 찾기 위해서는 for문이 더 많이 필요하기 때문에 작성이 힘들어 질 수 있습니다
